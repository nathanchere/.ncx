# .ncx

Just my Linux box setup / bootstrap scripts / dotfiles / etc

### Install

    git clone https://github.com/nathanchere/.ncx.git ~/.ncx
    cd ~/.ncx
   
    chmod +777 *.sh

    ./init_dotfiles.sh
    ./init_fonts.sh
    
    ./install_[whatever].sh
